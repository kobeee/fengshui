# 共享资源目录

此目录存放跨关卡共享的游戏资源。

## 目录结构

```
shared/
├── items/                 # 道具素材（透明背景 PNG）
│   ├── gourd.png         # 葫芦
│   ├── plant-broad.png   # 阔叶绿植
│   ├── screen.png        # 屏风
│   ├── money-toad.png    # 金蟾
│   ├── bagua-mirror.png  # 八卦镜
│   └── lucky-cat.png     # 招财猫
└── vfx/                   # 特效素材
    ├── sha-spirit/       # 煞气动画帧
    └── cleanse-glow/     # 净化光效
```

## 道具 ID 对照表

| ID | 名称 | 解决的煞气类型 |
|----|------|--------------|
| `gourd` | 葫芦 | 横梁压顶 |
| `plant-broad` | 阔叶绿植 | 尖角煞 |
| `screen` | 屏风 | 门冲 |
| `money-toad` | 金蟾 | 财位问题 |
| `bagua-mirror` | 八卦镜 | 多种外煞 |
| `lucky-cat` | 招财猫 | 财运提升 |
