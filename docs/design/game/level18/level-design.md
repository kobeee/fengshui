## 设计说明

作为大师篇关卡，本关引入了 **大师级煞气**：

1. **天斩煞** (`sky_split`): 窗外高楼切割，需山海镇化解
2. **路冲煞** (`road_clash`): 大门正对直路，需石敢当化解
3. **反弓煞** (`reverse_bow`): 道路弧形弯向房屋，需绿植缓冲

这些大师级煞气在之前关卡中从未出现，玩家需要学习新的化解方法。
