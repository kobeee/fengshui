# Level 4 - 猫奴的客厅 (Cat Lover's Living Room)

## 关卡概述

| 属性 | 值 |
|-----|-----|
| **关卡 ID** | `level-4` |
| **关卡名称** | 猫奴的客厅 |
| **英文名称** | Cat Lover's Living Room |
| **主题** | 猫咪主题客厅，到处是猫爬架和玩具 |
| **氛围** | 阴雨天的下午，有些闷 |
| **难度** | 进阶 (Intermediate) |
| **煞气数量** | 2 个 |
| **预估时间** | 2-3 分钟 |

---

## 叙事背景

> 你是一位资深的"猫奴"，客厅几乎被猫咪占领——猫爬架、猫窝、猫玩具无处不在。
>
> 最近家里总是有异味，而且感觉财气不聚。朋友来访说进门就直通阳台，气场全跑了...
>
> 风水先生指出两个问题：穿堂煞和味煞。该如何化解？

## 煞气点配置

### 煞气 1: 穿堂煞（门冲）

| 属性 | 值 |
|-----|-----|
| **ID** | `sha-001` |
| **类型** | `door_clash` |
| **位置** | `{ x: 0.50, y: 0.60 }` |
| **半径** | `0.10` |
| **正确解法** | 放置屏风阻挡 |
| **正确道具** | `screen` |

```yaml
shaPoint:
  id: sha-001
  type: door_clash
  position: { x: 0.50, y: 0.60 }
  radius: 0.10
  title: "穿堂煞"
  description: "入户门与阳台门成一条直线，气流直进直出，无法藏风聚气。"
  correctItem: screen
  options:
    - id: opt-001-a
      label: "放置屏风阻挡"
      correct: true
    - id: opt-001-b
      label: "常年关闭窗帘"
      correct: false
    - id: opt-001-c
      label: "门口放地垫"
      correct: false
```

### 煞气 2: 味煞

| 属性 | 值 |
|-----|-----|
| **ID** | `sha-002` |
| **类型** | `smell_sha` |
| **位置** | `{ x: 0.75, y: 0.50 }` |
| **半径** | `0.08` |
| **正确解法** | 移开猫砂盆 + 放置金蟾招财 |
| **正确道具** | `money-toad` |

```yaml
shaPoint:
  id: sha-002
  type: smell_sha
  position: { x: 0.75, y: 0.50 }
  radius: 0.08
  title: "味煞"
  description: "猫砂盆放在财位附近，秽气影响财运。应该移到通风处，财位放招财摆件。"
  correctItem: money-toad
  options:
    - id: opt-002-a
      label: "喷空气清新剂"
      correct: false
    - id: opt-002-b
      label: "移开猫砂盆 + 放金蟾招财"
      correct: true
    - id: opt-002-c
      label: "用布盖住猫砂盆"
      correct: false
```

## 场景元素

### 必须包含的视觉元素

| 元素 | 描述 | 位置 |
|-----|------|------|
| 入户门 | 客厅入口门 | 一侧墙边 |
| 阳台门 | 落地玻璃门，与入户门成直线 | 对面墙 |
| 沙发 | L型沙发，有猫毛毯 | 客厅中央 |
| 茶几 | 玻璃茶几，上有猫玩具 | 沙发前 |

| 猫砂盆 | 半开放式猫砂盆（财位附近） | 对角线位置 |
| 电视柜 | 电视柜和电视 | 另一侧墙 |

### 可选装饰元素

- 散落的猫玩具
- 猫窝/猫床
- 猫粮碗
- 猫咪海报

## 教学要点

本关卡教学目标：

1. **穿堂煞知识点**：门对门导致气场流失
2. **味煞/财位知识点**：污秽物品不宜在财位
3. **组合煞气处理**：需要多个道具解决不同问题
