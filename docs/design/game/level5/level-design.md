## 教学要点

本关卡教学目标：

1. **道具复用强化**：葫芦和绿植都是复用道具
2. **尖角煞变体**：不只是书架角，展示柜角也算尖角煞
3. **电竞玩家共鸣**：贴近年轻玩家生活场景
