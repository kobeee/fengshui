## 叙事背景

> 你和合伙人一起创业，租了一间小型工作室。白板上写满了创意，桌上堆着咖啡杯和原型机。
>
> 最近团队总是意见不合，项目进展缓慢。合伙人说他背对着门坐得很不安...
>
> 风水顾问指出：你的座位背门而坐，而且承重柱的尖角直冲工作区。这是"背门煞"加"柱角煞"！

## 教学要点

本关卡教学目标：

1. **办公室风水入门**：介绍办公环境的风水问题
2. **背门煞知识点**：背后无靠的危害
3. **柱角煞知识点**：建筑结构柱的影响
4. **新道具教学**：龙龟作为靠山摆件
