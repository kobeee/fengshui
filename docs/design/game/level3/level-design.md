## 教学要点

本关卡作为入门续篇，教学目标：

1. **新道具教学**：介绍阔叶绿植，用于化解尖角煞
2. **尖角煞知识点**：尖锐桌角、柜角对着床/人的风水禁忌
3. **道具复用概念**：同一道具可解决相同类型的煞气
