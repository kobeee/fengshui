## 教程设计说明

Level 1 作为游戏的第一关，设计原则：

1. **单一焦点**：只有 1 个煞气点，玩家可以专注学习基本操作
2. **视觉明显**：横梁压顶是最容易被发现的煞气类型
3. **操作简单**：只需选择正确道具，不需要复杂的交互
4. **即时反馈**：解决后立即看到房间变暖

### 教程引导流程

1. **发现阶段**：罗盘旋转加速，引导玩家找到横梁上方
2. **选择阶段**：弹出选项，玩家选择"挂葫芦化解"
3. **解决阶段**：葫芦出现在横梁上，房间色调变暖
4. **完成阶段**：显示通关动画，进入 Level Select
