## 完整提示词

```text
Generate a pixel art sprite of a Feng Shui gourd (hulu) for an isometric puzzle game.

Object description:
- Traditional Chinese gourd (hulu) shape - two connected round sections
- Small red rope/string attached at the top for hanging
- Natural dried gourd color (warm brown/tan)
- Subtle golden glow effect around it (representing positive Qi)

Technical requirements:
- Isometric 45 degree angle, matching room perspective
- Transparent background (alpha channel)
- Hi-bit pixel art style, consistent with room background
- Max 32 colors in palette
- Clean pixel edges, no anti-aliasing

Size reference:
- When placed in room, should appear hanging from ceiling/beam
- Approximately 20-30 pixels tall
- Scale appropriately for the room's perspective

Optional variations:
- Frame 1: Normal state
- Frame 2: Subtle glow pulse (for animation potential)

Output:
- Single PNG with transparency
- If animation frames provided, export as separate files or sprite sheet
```

---

## 生成后处理

1. **背景透明化**: 确保背景完全透明
2. **尺寸校准**: 调整到适合游戏内显示的尺寸
3. **发光效果**: 可添加额外的发光层 PNG 用于动画

---

## 游戏内使用

- **放置位置**: 悬挂在横梁下方
- **放置动画**: 轻轻飘落并挂住
- **净化效果**: 横梁区域出现暖光晕
